# История России - Минималистичный веб-сайт

Современный минималистичный веб-сайт, посвященный истории России от Древней Руси до современности.

## Особенности дизайна

- **Минималистичный стиль**: Чистый, современный дизайн с акцентом на контент
- **Адаптивная верстка**: Отлично выглядит на всех устройствах
- **Современная типографика**: Использование шрифта Inter для лучшей читаемости
- **Интерактивные элементы**: Плавные анимации и hover-эффекты
- **Визуальная иерархия**: Четкая структура информации с использованием цветов и размеров

## Структура сайта

### Главная страница (index.html)
- Героическая секция с заголовком
- Сетка исторических периодов с изображениями
- Карточки для каждого периода с кратким описанием

### Страницы исторических периодов:
1. **Древняя Русь** (ancient-rus.html) - IX-XIII века
2. **Московское государство** (moscow-state.html) - XIII-XVII века  
3. **Российская империя** (russian-empire.html) - XVIII-начало ХХ века
4. **Советский период** (soviet-period.html) - 1917-1991 годы
5. **Современная Россия** (modern-russia.html) - 1991-настоящее время

## Технические особенности

- **HTML5**: Семантическая разметка
- **CSS3**: Современные стили с Flexbox и Grid
- **Google Fonts**: Шрифт Inter для лучшей типографики
- **Unsplash Images**: Высококачественные изображения для исторических периодов
- **Responsive Design**: Адаптация под мобильные устройства
